API Key de Anthropic

La función ✦ AI Analyze usa Claude para leer y analizar los archivos de cotización. Para que funcione desde tu Mac necesitás una API key propia de Anthropic. Se guarda solo en tu navegador — nunca sale de tu computadora.

¿Dónde consigo una API key?
1. Entrá a console.anthropic.com
2. Creá una cuenta o iniciá sesión
3. Andá a API Keys -> Create Key
4. Copiá la key y pegala acá arriba

Costo: cada análisis de archivo usa aproximadamente $0.01–0.03 USD dependiendo del tamaño del documento.
